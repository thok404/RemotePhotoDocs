# RemotePhotoFrame

![476](../../assets/images/Pasted-image-20260507140548.png)

RemotePhotoFrame은 프레임을 설정하고 이미지 표시를 조정하는 데 사용합니다.

> 이 컴포넌트는 사진을 표시하는 메쉬 오브젝트에 배치해야 합니다.
>
> 사진 표시용 메쉬는 UV가 정규화된 단일 평면을 권장합니다.
>
> 사진 표시용 메쉬는 이 프로젝트에 포함된 Shader를 사용해야 합니다.

## 기본 설정

| 파라미터 | 설명 |
|---|---|
| Orientation (방향) | 현재 Frame의 구도 방향을 설정합니다. 실제 Frame 상태에 맞춰 Landscape 또는 Portrait를 선택합니다. |
| Material Slot (머티리얼 슬롯) | 이미지를 표시할 머티리얼 슬롯을 0부터 세어 선택합니다. 머티리얼이 하나뿐인 경우 보통 변경할 필요가 없습니다. |
| Texture Property (텍스처 프로퍼티) | 머티리얼 Shader 안의 메인 텍스처 슬롯 이름입니다. 이 프로젝트 전용 Shader를 사용하므로 보통 `_MainTex` 그대로 둡니다. |

![287](../../assets/images/Pasted-image-20260507010741.png)

## 이미지 표시

| 파라미터 | 설명 |
|---|---|
| Default Texture (기본 텍스처) | 원격 링크 이미지를 불러오기 전에 표시되는 이미지입니다. 비워 두어도 동작에는 영향이 없습니다. |
| Fallback Texture (폴백 텍스처) | 원격 이미지 로드에 실패한 뒤 표시되는 이미지입니다. 비워 두어도 동작에는 영향이 없습니다. |
| Background Color (배경색) | `Contain`에서 남는 빈 가장자리와 `Box` 모드의 측면 채움에 사용하는 배경색입니다. |

## 맞춤과 투영

| 파라미터 | 설명 |
|---|---|
| Photo Fit Mode (사진 맞춤 모드) | 일반적인 이미지 맞춤 모드를 제공합니다. |
| Projection Mode (투영 모드) | 사진 텍스처의 투영 방식을 선택합니다. |
| Horizontal Flip (수평 반전) | `Box` 투영이 좌우 반전되어 보일 때가 있습니다. 이 옵션으로 수정합니다. |
| Photo Rotation Offset (사진 회전 보정) | 사진이 최종 표시되기 전에 추가 회전을 적용합니다. 문제 수정이나 특수 효과가 필요하지 않다면 보통 0으로 둡니다. |

### Photo Fit Mode

| 옵션 | 설명 |
|---|---|
| Crop | 프레임 밖으로 넘친 부분을 잘라내며, 남은 이미지 비율은 바꾸지 않습니다. |
| Contain | 이미지 전체가 프레임 안에 들어가도록 크기를 조절하며 빈 가장자리가 남습니다. |
| Stretch | 이미지 비율을 무시하고 프레임을 가득 채우도록 늘립니다. |
| Tile | 원본 이미지 크기를 유지하고 프레임 안에서 패턴을 반복합니다. |

### Projection Mode

| 옵션 | 설명 |
|---|---|
| Mesh UV | 단일 평면 메쉬에 적합합니다. 사진 표시용 메쉬로 단일 평면을 권장합니다. |
| Box | 큐브 메쉬에 적합합니다. 큐브의 가로, 세로, 깊이를 감지하고 가장 짧은 변을 제외한 뒤, 나머지 두 치수로 투영 평면을 만듭니다. 실제 사용 시에는 사진 표시용 메쉬를 프레임처럼 납작한 큐브로 조정하면 됩니다. |

> 다른 형태의 지오메트리는 테스트되지 않았습니다. 단일 평면 또는 6면 큐브 메쉬만 사용하세요.

## 비율 계산

| 파라미터 | 설명 |
|---|---|
| Aspect Mode (비율 모드) | 이미지 표시 비율 계산 방식을 전환합니다. |
| Reference Box Axis Mode (Reference Box 축 모드) | Reference Box 모드에서 기준 박스로 비율을 계산하는 방식을 선택합니다. |
| Resolved Aspect Ratio (최종 비율) | 최종 적용되는 비율 값을 표시합니다. |

### Aspect Mode

| 옵션 | 설명 |
|---|---|
| Manual | `Manual Aspect Ratio`에 입력한 비율을 강제로 사용합니다. 필요한 비율을 정확히 알고 있을 때만 사용하세요. |
| Auto | 메쉬 자체의 경계에서 비율을 자동 계산합니다. 보통 이 옵션을 권장합니다. |
| Reference Box | BoxCollider처럼 편집할 수 있는 박스를 제공합니다. 이 박스로 프레임을 감싸면 됩니다. |

### Reference Box Axis Mode

| 옵션 | 설명 |
|---|---|
| Auto | 큐브 투영 계산처럼 가장 짧은 변을 제외하고, 남은 두 변으로 비율을 계산합니다. |
| Manual Axes | 어떤 두 축을 사용해 비율을 계산할지 수동으로 지정합니다. |
