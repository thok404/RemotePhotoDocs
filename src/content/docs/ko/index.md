사진이 너무 많고 맵 공간이 부족한가요? UV, 머티리얼, 텍스처를 조정하는 과정이 짜증나나요?
프레임이 인터넷에서 이미지를 직접 다운로드하게 해보세요


**RemotePhotoSystem**은 U# 컴포넌트 세트로, 외부 이미지 직접 링크를 저장하고 그것들을 사용해 갤러리를 만들 수 있습니다. 이를 통해 VRChat에서 적은 수의 프레임으로 모든 사진을 동적으로 로드할 수 있습니다. 컴포넌트는 프리로드, 랜덤/순서 사진 변경 모드, 적응형 프레임 비율 등의 기능도 제공해 경험을 더 부드럽게 만듭니다.


# 시작 전 알아둘 것

이미지 직접 링크 URL을 여러 개 직접 준비해야 합니다. 직접 링크는 보통 `.jpg`, `.png`, `.gif`, `.webp` 같은 이미지 형식으로 끝납니다

VRChat 필수 요구 사항: https://creators.vrchat.com/worlds/udon/image-loading/
이미지 최대 해상도는 2048 × 2048 픽셀입니다
5초마다 이미지 한 장을 다운로드할 수 있습니다
URL은 이미지 파일을 직접 가리켜야 하며, URL 리디렉션은 허용되지 않습니다
