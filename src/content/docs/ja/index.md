写真が多すぎてマップ容量が足りませんか？UV、マテリアル、テクスチャを調整する作業にうんざりしていますか？
フレームにインターネットから直接画像をダウンロードさせてみてください


**RemotePhotoSystem** は U# コンポーネント一式で、外部画像の直リンクを保存し、それらを使ってギャラリーを作成できます。これにより、VRChat で少数のフレームを使ってすべての写真を動的に読み込めます。コンポーネントはさらに、プリロード、ランダム/順序切り替えモード、自動適応フレーム比率などの機能を提供し、体験をよりスムーズにします。


# 始める前に

画像直リンク URL をいくつか自分で用意する必要があります。直リンクは多くの場合、`.jpg`、`.png`、`.gif`、`.webp` などの画像形式で終わります

VRChat の必須要件：https://creators.vrchat.com/worlds/udon/image-loading/
画像の最大解像度は 2048 × 2048 ピクセルです
5秒ごとに1枚の画像をダウンロードできます
URL は画像ファイルを直接指している必要があり、URL リダイレクトは許可されません
