***RemotePhotoFrame***用作设定相框和调整图像显示。
**==\*此组件需要挂载到显示图片的网格物体上\*==   
==\*显示图片用的网格物体推荐使用UV规格化的单平面\*==**
**==\*显示图片用的网格物体需要使用本项目自带的Shader\*==**

## 参数

**Orientation (图片方向池):** 设定当前相框的构图方向，有横向和纵向两个方向，根据相框的实际状态进行选择
![287](Pasted%20image%2020260507010741.png)

**Material Slot(材质槽位) :** 选择显示图像的材质槽位，从0开始数，通常只有一个材质的情况下不需要变动
**Texture Property (贴图属性名):** 材质球中的主色贴图槽在Shader中的命名，因为需要使用本项目自带Shader，通常此项保持默认_MainTex

**Default Texture (默认贴图):** 从远程链接加载图片前，相框所显示的图片，此项留空不影响程序运行
**Fallback Texture (回退贴图):** 从远程链接加载图片失败后，相框所显示的图片，此项留空不影响程序运行
**Background Color (背景颜色):** 设定一个背景颜色，用于Contain Fit Mode留下空白边缘的颜色， Box Projection Mode模式侧面填充

**Photo Fit Mode(图片适配模式) :** 提供了4种常见图片适配模式
	**Crop :** 将超出相框的部分裁切掉，不改变剩下的图片比例
	**Contain :** 将图片缩放到可以被相框完整容纳，会留下空白的边缘
	**Stretch :** 不考虑图片比例，强制拉伸图片填满相框
	**Tile :** 不改变原图大小，在相框内重复图案进行平铺

**Projection Mode (投射模式):** 选择照片贴图投影方式。
	**Mesh UV :** 适用于单平面网格，推荐使用单平面作为显示图片的网格物体，推荐使用此模式
	**Box :** 适用于立方体网格。此模式会自动检测网格立方体的长宽高，舍弃其中最短的，用剩余的两个构建投影平面，所以在使用时把显示图片的网格物体凭直觉调整为相框样式的扁平立方体就可以。
		**Horizontal Flip (水平翻转):** 有时Box投影会产生水平翻转的结果，激活这个选项修复它
	
	**其余几何形状未测试，请仅使用单平面或6面的立方体网格**

**Photo Rotation Offset (图片旋转补偿):** 相片最终呈现前进行一次额外旋转，通常保持为0，除非你需要修复一些错误或制作一些特殊效果。

**Aspect Mode (画幅比例模式):** 切换图片显示比例模式
	**Manual :** 将自定义数值填入Manual Aspect Ratio强制使用此比例，仅当你知道你自己在做什么时使用此选项
	**Auto :** 用网格自身的边界自动计算比例，**通常推荐使用**
	**Reference Box :** 你会获得一个编辑方式类似BoxCollider的盒子，用这个盒子包围你的相框即可
		**Reference Box Axis Mode**(Reference Box 轴向模式) 
			**Auto :** 像计算立方体投影模式一样舍弃最短的边长，用剩余的两个边长计算比例
			**Manual Axes :** 手动设定使用哪两个轴向的边计算比例

**Resolved Aspect Ratio (最终宽高比):** 最终应用的比例数值显示在这里
